# Lista 01 - Introdução a HTML e CSS

## Objetivo
Reforçar os conceitos fundamentais de HTML e CSS, incluindo estrutura básica de páginas, HTML semântico, seletores CSS e Box Model.

## O que foi desenvolvido

### Exercício 1
Criação da estrutura completa do HTML5 com definição de idioma, charset, título principal e parágrafo.

### Exercício 2
Organização do conteúdo utilizando tags semânticas: header, nav, main, section e footer.

### Exercício 3
Aplicação de CSS externo com uso de seletores de tag, classe e ID.

### Exercício 4
Criação de uma div com classe "card" demonstrando o uso do Box Model (width, padding, border, margin e box-sizing).

### Exercício 5
Organização adequada dos arquivos, uso de CSS externo e boas práticas de indentação.
